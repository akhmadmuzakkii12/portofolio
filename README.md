## ✨ Features

- 🖱️ **Custom animated cursor** with color trail effect
- 🌌 **Interactive particle system** that reacts to mouse movement
- ⌨️ **Typing animation** cycling through roles
- 🎴 **Auto-sliding project cards** — pauses on hover, supports drag & touch
- 📅 **Vertical timeline** for education & experience
- 📊 **Animated skill bars** triggered on scroll
- 📬 **Contact form** with animated submission
- 💡 **Glitch effects** & parallax on hero section
